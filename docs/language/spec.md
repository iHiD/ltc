# Langauge Specification

## Data Types

### Booleans

- `true`
- `false`

### Numbers

All double-precision floating points.

- `0`
- `1`
- `1.5`

## Expressions

### Arithmetic

- Addition: `1 + 2`
- Subtraction: `1 - 2`
- Division: `1 / 2`
- Multiplication: `1 * 2`

### Comparison

- Equality: `1 == 2` (`false`)

## Statements

### Print:

Printing to the console is via `print`.
This is not exposed in the UI.
